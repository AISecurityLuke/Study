This directory contains the 3 data sets combined into 1 in combined.jsonl

---------
dataset1: hackaprompt.parquet: supplied the malicious prompts
dataset2: ChatGpt.com: supplied the benign prompts
dataset3: ChatGpt.com: supplied the suspicious prompts
---------

combined.jsonl was created by combining all of the .jsonl files that were generated in a chatgpt chat.

I will use the same methods when I need larger data sets.
